# 6. Защищённые ветки

| Ветка       | Разрешённое действие | Исполнитель                    |
|-------------|----------------------|--------------------------------|
| `main`/`master` | Merge                | Релиз-менеджер                 |
| `develop`   | Merge                | Релиз-менеджер или архитектор  |
| `release/*` | Merge                | QA и релиз-менеджер            |
| `hotfix/*`  | Merge                | Релиз-менеджер                 |
